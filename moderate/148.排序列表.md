# 148.排序列表

## 题解

```python
class Solution:
    def sortList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        return self.sortFun(head)
        
    def sortFun(self, head: Optional[ListNode]):
        if not head or not head.next:
            return head
        
        slow = head
        fast = head.next
        while fast and fast.next:
            slow = slow.next
            fast = fast.next.next
        
        new_head = slow.next
        slow.next = None
        return self.merge(self.sortFun(head), self.sortFun(new_head))

    def merge(self, l1: Optional[ListNode], l2: Optional[ListNode]):
        if not l1:
            return l2
        if not l2:
            return l1

        dummy = ListNode(-1)
        cur = dummy

        while l1 or l2:
            if not l1:
                cur.next = l2
                break
            if not l2:
                cur.next = l1
                break

            if l1.val < l2.val:
                cur.next = l1
                l1 = l1.next
            else:
                cur.next = l2
                l2 = l2.next

            cur = cur.next
            
        return dummy.next
```

# 解析

这道题可以用递归的思想来解决，考虑到对于一个链表，从中间切开，分别将它们排序，再使用[21.合并两个有序链表](../easy/21.合并两个有序链表.md)的方式合并成一个有序链表，为了将它们排序，可以再次切割排序并合并，这样就得到了递归的思路，只有当需要排序的链表是空或者只有一个节点时直接返回
